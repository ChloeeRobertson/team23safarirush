About Team 23

Purpose


This site was created for the COMP2910 project course at BCIT by 2016 CST students.


The main objective was to create a brain game that works on mobile web browsers, while learning software project management and development practices.


Our goal was to create a simple, intuitive, and addictive puzzle solving game for everyone!
